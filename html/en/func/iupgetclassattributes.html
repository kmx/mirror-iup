IupGetClassAttributes

Returns the names of all registered attributes of the class of an element.

Parameters/Return

int IupGetClassAttributes(Ihandle* ih, char *names[], int max_n); [in C]
iup.GetClassAttributes(ih: ihandle, max_n: number) -> (names: table, n: number) [in Lua]

ih: identifier of the interface element.
names: table receiving the names
max_n: maximum number of names the table can receive.

Returns the number of names loaded to the table.

See Also

IupGetClassName, IupGetClassType, IupGetAllAttributes
