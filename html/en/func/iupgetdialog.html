IupGetDialog

Returns the identifier of the dialog that contains that interface element. Works also for children of a menu that is associated with a dialog.

Parameters/Return

Ihandle* IupGetDialog(Ihandle *ih); [in C]
iup.GetDialog(ih: ihandle) -> (ih: ihandle) [in Lua]

ih: Identifier of an interface element.

Returns NULL if not found.
